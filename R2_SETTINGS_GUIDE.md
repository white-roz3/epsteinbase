# R2 Settings Tab - What to Do

## In the R2 Bucket Page:

1. **Click the "Settings" tab** (next to "Objects" and "Metrics")

2. **In Settings, look for**:
   - "API Tokens" section
   - "Manage R2 API Tokens" link/button
   - Or scroll down to find token management options

3. **Alternative**: Look for "Manage R2 API Tokens" button at the top-right of the R2 dashboard (outside the bucket page)

Once you're in the API Tokens section, you'll see:
- "Create API Token" button
- Options for:
  - **Token Name**: `epsteinbase-api`
  - **Permissions**: **Admin Read & Write**
  - **Bucket**: `epsteinbase`

Click "Settings" tab and let me know what you see!
